      Boolean数据类型 #

    • 转换规则：
      • 如果只有一个值，判断这个值的真假；
      • 0,NaN,"",undefined,null这五个值为假
      • val1 == val2 两值相比若不是同一数据类型的，先默认进行数据转换
      • 对象==对象 永远不相等
      • 对象==字符串 先将对象转换为字符串(toString())，然后再进行比较
        • [].toString(); ==> ""
        • ({}.toString();) ==> [object Object]
      • 对象==布尔类型 对象先转换为字符串(toString())，字符串再转换为数字(Number());布尔类型转换为数字(true:1,false:0);最后两数字比较
        console.log([]==false);
      • 对象==数字 对象先转换为字符串(toString())，字符串再转换为数字(Number())
      • 数字==布尔 布尔类型转换为数字(true:1,false:0)
      • 数字==字符串 字符串再转换为数字(Number())
      • 字符串==布尔 都转换为数字，然后比较
      • null == undefined true
      • null或undefined和其他任何数据类型相比都不相等
      • === 绝对比较 如果数据类型不一样，肯定不相等
      • Number(Null) //0
      • Number(undefined) // NaN -
        console.log(0 === false);
console.log(null === undefined);
    • !:取反（先将值转化为布尔类型，然后再取反）
    • !!:将其他数据类型转换为boolean类型，相当于Boolean();

//获取属性名和属性值

    数据类型区分和数据类型检测 #

    • 基本数据类型和引用数据类型的区别
      • 基本数据类型把值直接赋给变量，直接对值进行操作
      • 引用数据类型开辟一段内存，将属性名和属性值保存在里面，并拥有一个内存地址
      • 变量引用的是内存地址，并没有存储数值 基本数据类型操作的是值；引用数据类型操作的是引用内存地址
    • 检测数据类型方式
      • typeof运算符，返回值：字符串，包含"number","string","boolean","undefined","function","object"
      • typeof不能具体检查object下细分的数据类型，如:数组，对象，正则...
        console.log(typeof typeof typeof typeof []);
      • instanceof运算符
      • constructor
      • Object.prototype.toString.call()

    for循环 #

    • 第一步，设置初始值 var i=0;
    • 第二步，设置循环执行条件 i<5;
    • 第三步，执行循环体中的内容{...}
    • 第四步，每一轮循环完成后执行i++操作
    • break:在循环体中出现后，整个循环立即结束
    • continue:在循环体中出现后， 当前这轮循环停止，继续下一轮循环
    • 在循环体中遇到这两个关键字，循环体后面的代码就不执行了

    • for(var key in obj){key,obj[key]}

    • 循环顺序：若有数字为键值，则按照数字顺序由小到大循环，剩下的再按照对象中的顺序循环

    function #

    • function:函数数据类型
    • 定义函数的步骤：
      • 1.开辟一段新的空间地址
      • 2.把函数的js代码以字符串存在空间中
      • 3.调用
    • 作用：实现一个功能的代码的封装
    • 形参
      • 作用：实现一个功能，把需要的原材料定义到形参上，在执行时提供；
      • 一个功能的多种形态
      • 若定义了形参而执行时未赋值，则为undefined
    • arguments:函数内置接收参数机制
      • console.dir()输出信息更详细
      • 类数组，有数字索引，从0开始；具有length属性，代表参数长度
      • arguments.callee代表当前函数本身
      • 获取某一项参数:arguments[i]
    • 闭包：函数执行时形成的私有作用域，来保护里面的变量不受外界干扰
    • 若外界想使用函数执行结果，则return这个值
    • 若没有写return则默认返回结果为undefined;若只写return而没写变量，则为undefined;
    • 函数return 之后，后面的语句不再执行
    • 实名函数，匿名函数
    • 函数表达式，把一个匿名函数的定义部分作为变量或元素事件
    • 自执行函数：定义和执行一起完成
      (function(){
//定义部分
})();  //执行部分

        数组和数组的常用方法(12) #

    • 创建数组的两种方式
      • var arr = [1,2,3,4];
      • var arr = new Array();
    • 查看数组的详细信息：console.dir(arr)
    • 数组也是对象数据类型，由属性名和属性值组成
    • arr.length获取数组的长度
    • 数组的 增加 删除 修改
    • arr.push(16); -向数组末尾添加元素，返回值为新数组的长度；原有数组改变了
    • arr.unshift(); -向数组的开头添加元素，返回值为新数组的长度；原有数组改变了
    • arr.splice(n,0,x); -向数组中某个位置添加元素，从索引n开始，删除0个内容，并在索引n之前添加内容x；返回的是一个空数组；原有数组改变
    • arr.splice(n,m); -从索引n开始(包含n)，删除m个元素，并将删除的内容作为数组返回
    • arr.splice(n,m,x); -把原数组中的某些项进行替换，从索引n开始删除m个元素，用x替换原来的，并返回删除的内容
    • arr.splice(0,0,x)相当于arr.unshift(x)，在数组开头添加元素
    • arr.splice(arr.length,0,x)相当于arr.push(x)，在数组末尾添加元素
    • arr.pop(); -删除数组最后一项，并将删除的元素作为返回值，原有数组长度减一

    • arr.shift(); -删除数组第一项，并将删除的元素作为返回值，原有数组长度减一

    • 数组的 查询 复制

    • arr.slice(n,m); -从索引n开始(包含n)，找到索引m处(不含m)，将找到的内容作为一个新的数组返回，原有数组不变
    • splice和slice的区别在于对原数组的改变
    • arr.slice(n); -从索引n开始(包含n)，找到数组末尾
    • arr.slice(0)或arr.slice() -将原有数组复制一份，属于数组的克隆
    • arr.concat() -实现数组的克隆，原来的数组不变，相当于slice(0)

    • arr.concat(arr2)本意用于数组的拼接,将数组arr和arr2拼接成一个数组；返回拼接后的数组

    • 数组转化为字符串

    • toString() -[] == "" //true 把数组中每一项用“,”隔开，连接为字符串，原有数组不变
    • join(分隔符) -把原有数组按照指定分隔符连接成字符串

    • join("+")、eval()将指定的字符串变成真正的表达式执行

    • 数组的 排列 排序

    • reverse() -将数组倒过来排列，原有数组改变
    • sort() -将数组由大到小或由小到大排列，原有数组改变；直接用sort排序只能处理10以内的排序；

    • sort(function(a,b){return a-b;}) -10以上的需要传递参数，且用函数作为参数;a代表每次循环的当前项，b代表后面的项

    • 数组常用但不兼容的方法

    • indexOf(); 判断数组是否包含某一项
    • forEach(); 循环数组中每一项

    • map(); 循环每一项，操作参数

    • 数组的冒泡排序，快速排序，插入排序

    • 当前项和后一项进行比较，若当前项大于后一项，则交换位置

    函数数据类型 #

    • 创建函数，function
    • 执行函数
    • 当浏览器加载页面的时候，首先会提供一个供js运行的环境(全局作用域)
      • 创建一个函数，首先开辟一段内存空间，浏览器为其分配一个16进制的地址
      • 把函数中的代码以字符串的形式存储到空间内
      • 再把当前的地址赋值给函数名
    • 函数名只代表函数本身
    • 函数名后带小括号则代表执行当前函数
    • 函数执行步骤：
      • 函数执行时，会为js代码提供一个私有作用域
      • 把字符串变为js代码，并从上至下执行
      • 私有作用域保护了里面的变量不受外界干扰，这种形成的保护机制叫闭包
    • 形参
    • 返回值return:当前函数提供的出口；若不提供出口的话，由闭包的机制导致了在函数体外面无法获取里面变量的值
    • return 并不是把变量返回给外边，而是把变量的值返回到外边
    • 若函数没有返回值，则默认返回值为undefined
    • arguments函数接收参数的集合

    • 数据类型为对象；以数字作为索引，0开始；有length属性；不是数组，而是类数组；它还有自己的特殊属性callee,代表函数本身

    • 常用匿名函数方式:

      • 函数定义自执行
        ;(function(num){})(100);  //加分号的作用是防止前边的函数结束时不加分号
      • 函数表达式
